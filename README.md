# API-Hackathon


Reglas
● Al menos 2 participantes por equipo y máximo 3
● Código en Github (debe ser público)
● Deb ocupar al menos 2 APIs de Bancos, Fintechs, Reguladores, Sistemas de
Información Financiera.
